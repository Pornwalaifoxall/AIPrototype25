# 🧠 ภาพรวมของโปรเจกต์

- คำแนะนำสำหรับผู้อ่าน

# 🌐 ความสามารถของระบบ

### 📰 การตรวจจับข่าวปลอม

### 🔎 การค้นหาหลักฐาน
ค้นหาข้อมูลสนับสนุนจากอินเทอร์เน็ต

### 🧠 การอธิบายผลลัพธ์

### 📚 แสดงแหล่งอ้างอิง
