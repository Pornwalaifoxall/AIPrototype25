# 🧠 ภาพรวมของโปรเจกต์

**TH HEALTH LENS** เป็นระบบปัญญาประดิษฐ์ที่ถูกพัฒนาขึ้นเพื่อช่วย **ตรวจสอบความถูกต้องของข่าวถาษาไทยในหมวดสุขภาพ**

ระบบสามารถช่วยผู้ใช้วิเคราะห์ว่าเนื้อหาข่าวนั้นเป็น

- ✅ **ข่าวจริง**
- ❌ **ข่าวปลอม**
- ⚠️ **ข่าวที่ต้องตรวจสอบเพิ่มเติม**

พร้อมทั้งแสดง

- เหตุผลในการวิเคราะห์
- แหล่งข้อมูลอ้างอิง
- คำแนะนำสำหรับผู้อ่าน

# 🌐 ความสามารถของระบบ

### 📰 การตรวจจับข่าวปลอม
วิเคราะห์เนื้อหาข่าวสุขภาพด้วย AI

### 🔎 การค้นหาหลักฐาน
ค้นหาข้อมูลสนับสนุนจากอินเทอร์เน็ต

### 🧠 การอธิบายผลลัพธ์
อธิบายเหตุผลในการวิเคราะห์ข่าว

### 📚 แสดงแหล่งอ้างอิง
แสดงแหล่งข้อมูลที่เกี่ยวข้อง

### 💡 คำแนะนำสำหรับผู้ใช้
ช่วยให้ผู้ใช้เข้าใจข่าวอย่างถูกต้อง
